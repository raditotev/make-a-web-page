Summary

1. The Basics of the Web

World Wide Web

WWW is a collection of HTML documents. Hyper Links connect between web pages.

Components of the Web

The three major pieces of the web are the user, the Internet and the servers. By using HTTP requests user's browser gets files back from the servers.

2. HTML

What is HTML?

HTML documents are the heart of the web. HTML stands for HyperText markup language. HTML is made up of text content (what you see), markup(what that content looks like or how it's arranged), references to other documents (images, videos etc.) and links to other pages.

Tags

HTML markup is made up of things called opening and closing tags. The whole thing might be referred as an element. There are two types of elements: inline and block elements. Inline element is in line with the text (e.g. span), whereas block element forms an imaginary block around the content (e.g. div). There are also void elements such as: br, img and link.

Marking Links

There is a special type of tags with attributes. These are tags withholding attributes inside. Examples of tags with attributes are img, a, link etc. Creating links to other content is done using the src (for images) or href attributes.

Structure

The basic structure of an HTML document consist a head element containing title element and body, all wrapped in html element. As the HTML page grows there will be more elements within element or so called tree like structure.

3. Creating Structured Documents

First Step

Building a web page can be compared with the building of a house. We first start building the foundations, the walls and we finish the basic structure with the roof. This is just like the HTML. After we finish with the basic structure we start work on the external and internal decoration of the house. Painting the walls, furnishing the place and hanging paintings. This is the role of CSS in our document. With CSS we can set different appearance for every single element in our HTML document. Finally we equip our house with some gadgets. Some might might be as simple as automatic garage door some might be as complex as home automation system. This is the place for Javascript in our web page.

HTML-CSS-DOM

HTML is the language of the web. It has syntax and its rules. The basic word in HTML is tag. The browser turns HTML tags into elements that form a tree. It knows how to do that because of the DOM.

CSS - Cascading Style Sheets - style sheet language used for describing the look and formatting of a document written in a markup language.

DOM - Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

Boxes, Grids and Rules

Everything on a web page is a box. By thinking in terms of boxes front end developers can easily rearrange things if they decide to give the users different experience. There is also a grid in the background that serves to make sure that everything is properly aligned.

Boxes are made up of several components. The first one is the core of the box and this is the content. It might be text or image or video or something else. Than we have padding, surrounded by the border and the outer component is called margin.

4. CSS Style and HTML Structure

Adding Style

Adding style to a html document can be done in different ways. We can link our HTML document to a CSS document using the link tag in the head element. We can also add style tag within the HTML document in the head element and set the appearance for the different elements or we can use styling attributes within the individual elements.

Positioning Boxes

If we want to put two boxes next to each other we have to do some positioning. To do so we're using the flex box layout. In order to achieve our goal we have to apply this style to the parent container. Than all child containers will align in a row.
